Design Guidelines

The Hedwig Design System reflects the patterns and components that underpin all of the ProctorU service lines. These colors are intended to be WCAG 2.0 compliant and will visually offset static content.

Primary Blue

Hex: #0072b1

RGB: 0/114/177

Contrast Ratio

when used with light colors

5.19:1

Contrast Ratio

when used with dark colors

2.43:1

Primary Blue should be the main color used in any digital project. This blue should be used for Headings of content sections, main action buttons, primary iconography or illustration, or anywhere that is crucial for navigation or userflow. If primary blue is used as the background color then the text color should ALWAYSbe light to provide enough contrast.

Primary blue should NEVER be used as regular text color, ther isnt enough color contrast between the background and foreground.

Primary blue should NEVER be used on a dark background, there isnt enough color contrast between background and foreground.
