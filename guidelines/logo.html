Design Guidelines

Digital Use Logo

Primary Logo should should use appropriate padding to give enough separation between the logo and the content. Digital Use logo is comprised of two differnt elements; the logo type and the owl mark.

Primary Logo should should follow these guidelines:

  • Logo Type should not be used without the owl mark.
  • Logo Type should always provide enough contrast with the background to be WCAG AA compliant.
  • Primary Digital Logo should always include the trademark symbol.

Primary Logo Print Use

Primary Logo should should use appropriate padding to give enough separation between the logo and the content. Print Use logo is comprised of two differnt elements; the logo type and the owl mark.

Primary Logo should should follow these guidelines:

  • Logo Type should not be used without the owl mark.
  • Owl Mark should use ProctorU Blue whenever applicable.
  • Primary Digital Logo should always include the trademark symbol.

Secondary Logo Use

Secondary Logo should use appropriate padding to give enough separation between the logo and the content. Secondary Use logo is only the owl mark.

Secondary Logo should should follow these guidelines:

  • Print Use Secondary Logo should be used in ProctorU Blue Whenever applicable.
  • Digital Use Secondary Logo should be used in the WCAG AA Blue when applicable.
  • Secondary Logo should always provide enough contrast with the background to be readable.
  • Secondary Logo should always include the registered trademark symbol.

Logo Alternate Use

Monochrome and grey scale usage of all ProctorU Logo's is available, but should still follow the best practices for the logo being used.

Logo Standards

The ProctorU brandmark should never be altered from the official standards. Do not change fonts, spacing, ratios or color on any brandmark element. Additionally, refrain from the use of drop shadows with the ProctorU brandmark.
