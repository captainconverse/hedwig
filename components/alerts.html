Components Overview

Alerts

Alerts should follow these general guidelines

  • Accessibility should always be a top consideration. Text color and link color pass WCAG AAA standards for color contrast
  • Text should always be left aligned
  • Border-radius should never exceed 4px, however is alert spans 100% of the parent container then no radius should be aplied
  • Alerts can be in any of the Brand colors and can accept bot links, headings, list items, and icons
  • Alerts should have a minimum of 4px of internal padding
