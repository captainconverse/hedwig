ProctorU Definitions

General Definitions

Proctor
American Usage. A supervisor especially of an examination or dormitory in a school.
Invigilator
European Usage. A supervisor especially of an examination or dormitory in a school.
Procki
The mascot for ProctorU

General User Roles

Test-taker
The person that is actually taking the test, formerly "student". Student and test-taker should not be used interchangeably since proffesional testing organizations do not identify as a student.
Organization
should only be used to define profesional testing centers
Institution
should only be used to define k12 - higher education testing centers
Prospect
Any user that doesn't curently work with ProctorU but is somewhere in the sales process
Institution User Editor
Fewest account privilages. This role gets assigned during implementation. This role can only add or edit institution/organization user accounts.
Institution Exam Administrator
User with the ability to add, upload, and edit information in ProctorU system only at exam level. This user needs to provide exam information, but not look at any other meaningful data.
Institution Instructor (Auto-Only)
User is only able to VIEW exams associated with themselves in ProctorU, but can create new exams via ProctorU Auto process.
Institution Instructor Observer
User with the ability to add, upload, and edit information in ProctorU system only at exam level. This user needs to provide exam information, but not look at any other meaningful data.

Product Definitions

ProctorU Auto
Fully-automated platform with multi-factor identity verification and end-to-end video recording with AI-based behavior flagging. Great for lower to mid-stakes exams or quizzes.
Live+
Professionally-trained live proctors supported by AI-based behavior analysis and intervention specialists. A good option for any online exam.
Pro
Our most-experienced live proctors augmented by AI, customized to the clients needs. Perfect for professional testing and certifications in every industry.
Groups
Our most-experienced live proctors augmented by AI, customized to the clients needs. Perfect for professional testing and certifications in every industry.
AuthBeta
An identity management solution with continuous biometric authentication for as long as the user requires verification. Can be paired with proctoring services or stand alone.
Archimedes
